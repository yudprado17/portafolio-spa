## 🛠️ Tecnologías Utilizadas

Este proyecto fue construido con un enfoque en rendimiento y diseño moderno:

- **Frontend Core:** [React 19](https://react.dev/) (Última versión)
- **Herramienta de Construcción:** [Vite](https://vitejs.dev/) para un desarrollo ultrarrápido.
- **Estilos:** [CSS3 Vanilla](https://developer.mozilla.org/es/docs/Web/CSS) con arquitectura de variables personalizadas y [Bootstrap 5](https://getbootstrap.com/) para el layout.
- **Iconos:** [React Icons](https://react-icons.github.io/react-icons/)
- **Compartir:** [QR Code React](https://github.com/zpao/qrcode.react) para generación dinámica de códigos QR.

## ✨ Características Principales

- 🌓 **Modo Claro/Oscuro dinámico**: Personalización completa basada en las preferencias del usuario.
- 📱 **Diseño Totalmente Responsivo**: Optimizado para móviles, tablets y escritorio.
- 🎢 **Carruseles Interactivos**: Para una visualización elegante de certificados y testimonios.
- 🖼️ **Galería con Modales**: Zoom de imágenes para certificados y proyectos.
- 📬 **Formulario de Contacto**: Integrado y estilizado.
- 🔗 **Generador de Código QR**: Permite a los usuarios compartir el portafolio fácilmente.

## 📂 Estructura del Proyecto

`	ext
src/
├── assets/         # Imágenes, iconos y recursos estáticos
├── components/     # Componentes React reutilizables (Hero, About, Projects, etc.)
├── context/        # Manejo de estado global (Tema, Idioma)
├── index.css       # Sistema de diseño centralizado y variables
└── App.jsx         # Componente principal de la SPA
`

## 🚀 Instalación y Ejecución Local

Si deseas clonar este proyecto y ejecutarlo localmente:

1. Clona el repositorio:
   `ash
   git clone https://github.com/yudprado17/portafolio-spa.git
   `
2. Entra a la carpeta:
   `ash
   cd "portafolio spa"
   `
3. Instala las dependencias:
   `ash
   npm install
   `
4. Inicia el servidor de desarrollo:
   `ash
   npm run dev
   `
